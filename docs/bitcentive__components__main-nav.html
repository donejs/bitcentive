<main-nav {(page)}="page" {(session)}="session"></main-nav>

Creates the main navigation

Parameters

  1. page {String}

    The name of the active page

  2. session {Session}

    The session of the app

Main navigation of the app

To create a <main-nav> element pass the Session and page:

<main-nav
    {(page)}="displayedPage"
    {(session)}="session"></main-nav>
